Playground — it actually runs

Evaluate an EML s-expression to a complex value. Grammar: S → 1 | <var> | (eml S S). Bind variables below.

Only 1 is a literal; every constant is a derived tree. ln works on negatives via the principal branch (you'll see Euler's identity emerge).

Check a Hehner predicate over a pre-state and a post-state. Post-state variables are primed (x'). Returns true/false.

Forms: true false (= a b) (and …) (or …) (not p) (pred E). Equality is exact.

The exact matmul-decomposition verifier (M5 Discovery, Phase 0). A scheme of {-1,0,+1} triples reconstructs the target tensor T_n; a discovery is registered only when the error is exactly 0.

Strassen's 7 multiplications reconstruct T_2 exactly — the result AlphaTensor searched for. The engine (R-0008) will search for such schemes at ever-smaller rank.
